What is Typeless.css?

Typeless.css is a revolutionary approach to web design that combines Swiss design precision with Japanese minimalism. It automatically styles every HTML element to create beautiful, professional-looking websites without requiring any CSS classes.

How to Use

1. Include the CSS

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/vnorman1/typeless.css@master/Typeless_min.css">

2. Write Semantic HTML

<!DOCTYPE html>
<html>
<head>
    <title>My Beautiful Website</title>
    <link rel="stylesheet" href="Typeless.css">
</head>
<body>
    <header>
        <h1>Welcome</h1>
        <p>This is automatically beautiful!</p>
    </header>
    
    <main>
        <article>
            <h2>Article Title</h2>
            <p>Your content here...</p>
        </article>
    </main>
</body>
</html>

3. That's It!

Your website is now beautifully styled with zero configuration. Add more HTML elements and they'll automatically look great!

Customization Options

While Typeless.css works perfectly out of the box, you can customize colors, fonts, and spacing by modifying the CSS custom properties at the top of the file.

:root {
  --color-primary: #2563eb;        /* Change accent color */
  --font-body: "Your Font", sans-serif;  /* Change font */
  --content-width: 65ch;           /* Adjust content width */
  /* ... more variables ... */
}
